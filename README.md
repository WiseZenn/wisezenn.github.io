## Quick Start

### Local Development

```bash
# Start development server
docker compose up
```

Visit http://localhost:4000 to preview the blog.

### Build & Deploy

```bash
# Build static files
./scripts/build.ps1

# Deploy to GitHub Pages
./scripts/deploy.ps1
```
